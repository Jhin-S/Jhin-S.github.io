## 1. 스택(Stack) 자료구조 개요

**[KR]** 스택(Stack)은 데이터를 보관하는 일종의 컨테이너(Container) 혹은 가방(Bag)과 같은 역할을 하는 추상 자료형입니다. 스택의 가장 핵심적인 특징은 나중에 들어간 데이터가 가장 먼저 나오는 **후입선출(LIFO: Last-In-First-Out)** 구조를 따른다는 점입니다. 

본 실습 코드에서는 C++ 템플릿(Template)을 활용하여 구현된 `Stack` 클래스를 테스트하며, `Stack<char>` 뿐만 아니라 주석 처리된 `Stack<int>` 처럼 다양한 자료형의 스택을 쉽게 생성할 수 있음을 보여줍니다.

---

## 2. 스택의 핵심 연산 및 주의사항

**[KR]** 제공된 테스트 코드는 스택의 기본적인 연산들이 정상적으로 동작하는지 확인합니다.

* **Push:** 스택의 최상단에 새로운 데이터를 밀어 넣습니다.
* **Pop:** 스택의 최상단에 있는 데이터를 꺼내어 제거합니다.
* **Top:** 스택의 최상단에 있는 데이터를 제거하지 않고 반환하여 확인합니다. 
* **주의사항:** 스택이 완전히 비어있을 때(Empty 상태) `Top()` 함수를 호출하는 것은 불가능하므로 예외 처리에 주의해야 합니다.

---

## 3. 테스트 코드 및 문자열 뒤집기 활용 (Implementation & Application)

**[KR]** 코드 후반부에는 스택의 LIFO 성질을 응용한 대표적인 알고리즘인 **문자열 순서 뒤집기** 실습이 포함되어 있습니다. "Hello, World!"라는 문자열의 각 문자를 순서대로 스택에 `Push`한 뒤, 스택이 빌 때(`IsEmpty()`)까지 `Pop`을 수행하며 `Top`을 출력하면 문자열이 역순으로 출력됩니다.

```cpp
#include <iostream>

#include "../shared/Stack.h" // 경로 "shared" 주의
using namespace std;
int main()
{

	// 일종의 가방(bag), 컨테이너(container)
	// LIFO(Last-In-First-Out)

	Stack<char> s;

	s.Push('A');
	s.Print();

	s.Push('B');
	s.Print();

	s.Push('C');
	s.Print();

	std::cout << "Top = " << s.Top() << std::endl;

	s.Pop();
	s.Print();

	std::cout << "Top = " << s.Top() << std::endl;

	s.Pop();
	s.Print();

	s.Push('E');
	s.Print();

	s.Pop();
	s.Print();

	std::cout << "Top = " << s.Top() << std::endl;

	s.Pop();

	// cout << "Top = " << s.Top() << endl; // 비었을 때 Top() 불가

	// 다른 자료형의 stack도 쉽게 만들 수 있어요.
	{
		//Stack<int> int_stack;
		//int_stack.Push(123);
		//cout << int_stack.Top() << endl;
		//int_stack.Top();
	}

	// 문자열 순서 뒤집어서 출력하기 (스택 사용하기 연습)
	{
		const char str[] = "Hello, World!";
		const int size = sizeof(str) - 1; // 마지막 '\0' 제외

		std::cout << "Input: " << str << std::endl;

		Stack<char> stack;

		// 문자열의 각 문자를 스택에 Push
		for (int i = 0; i < size; i++) {
			stack.Push(str[i]);
		}

		// 스택이 빌 때까지 Top을 출력하고 Pop 수행 (LIFO에 의해 역순 출력됨)
		while (!stack.IsEmpty()) {
			std::cout << stack.Top();
			stack.Pop();
		}

		std::cout << std::endl;
	}

	return 0;
}
```
