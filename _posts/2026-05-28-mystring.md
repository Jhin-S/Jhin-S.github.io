## 1. 문자열 추상 자료형 (String Abstract Data Type)

**[KR]** C++의 표준 라이브러리인 `<string>`을 사용하지 않고, 포인터와 동적 메모리 할당을 이용하여 문자열 클래스(`MyString`)를 직접 구현하는 과제입니다. 주어진 `main` 함수(테스트 코드)가 모두 정상적으로 동작하도록 헤더 파일과 소스 코드를 작성해야 합니다.

**[EN]** This task involves implementing a custom string class (`MyString`) using pointers and dynamic memory allocation, without relying on the C++ standard `<string>` library. The goal is to write the header and source files so that the provided `main` function (test code) executes correctly.

### 테스트 코드 (Problem: Test Code)
`MyString` 클래스가 정상적으로 구현되었는지 확인하기 위한 테스트 코드입니다. 생성자, 부분 문자열 탐색(`Find`), 복사 생성자, 문자열 비교(`IsEqual`), 삽입(`Insert`), 자르기(`Substr`), 연결(`Concat`) 기능이 올바르게 작동해야 합니다.

## 2. 헤더 파일 (MyString.h)

**[KR]** `MyString` 클래스의 구조를 정의한 헤더 파일입니다. 문자열의 길이를 저장하는 `str_size`와 동적으로 할당된 문자열 배열을 가리키는 포인터 `str_data`를 멤버 변수로 가집니다.

**[EN]** This is the header file that defines the structure of the `MyString` class. It contains `str_size` to store the length of the string and a pointer `str_data` that points to the dynamically allocated character array.

```cpp
// MyString.h의 코드

#pragma once

#include <iostream>

class MyString {
private:
	size_t str_size = 0;
	char* str_data = nullptr;

public:
	MyString(const char* init);
	MyString(const MyString& other);
	~MyString();

	void Print();
	int Find(const MyString& message);
	int Length();
	bool IsEqual(const MyString& other);
	MyString Insert(const MyString& other, int start);
	MyString Substr(size_t start, size_t keep);
	MyString Concat(const MyString& other);
};
```

## 3. 소스 코드 구현 (MyString.cpp)

**[KR]** 헤더 파일에 선언된 기능들을 실제로 구현한 소스 코드입니다. 새로운 문자열을 생성하거나 변형할 때마다 메모리를 동적으로 재할당(`new char[]`)하고, 필요 없는 메모리를 해제(`delete[]`)하는 과정을 수동으로 처리합니다. 깊은 복사(Deep Copy)를 통해 얕은 복사(Shallow Copy)로 인한 메모리 누수나 오류를 방지했습니다.

**[EN]** This is the source code where the functions declared in the header file are actually implemented. Whenever a string is created or modified, memory is dynamically reallocated (`new char[]`), and unused memory is manually deallocated (`delete[]`). Deep copy is utilized to prevent memory leaks and errors that can occur from shallow copying.

```cpp
//MyString.cpp의 코드

# include <iostream>
# include "MyString.h"
using namespace std;

MyString::MyString(const char* init) {
	while (true) {
		if (*(init + str_size) != '\0') {
			str_size++;
		}
		else {
			break;
		}
	}
	str_data = new char[str_size+1];
	for (size_t i = 0; i < str_size+1; i++) {
		*(str_data + i) = *(init + i);
	}
}

MyString::MyString(const MyString& other) {
	this->str_size = other.str_size;
	this->str_data = new char[this->str_size+1];
	for (size_t i = 0; i <= this->str_size; i++) {
		*(this->str_data + i) = *(other.str_data + i);
	}
}

MyString::~MyString() {
	if (str_data != nullptr){
		delete[] str_data;
	}
}

void MyString::Print() {
	for (size_t i = 0; i < str_size; i++) {
		cout << *(str_data + i);
	}
	cout << '\n';
}

int MyString::Find(const MyString& message) {
	size_t size_message = message.str_size;
	size_t start = 0;
	if (size_message > this->str_size) return -1;
	while (start <= this->str_size - message.str_size) {
		bool is_equal = true;
		for (size_t i = 0; i < size_message; i++) {
			if (*(this->str_data + start + i) != *(message.str_data + i)) {
				is_equal = false;
				break;
			}
		}
		if (is_equal == true) {
			return start;
		}
		else {
			start += 1;
		}
	}
	return -1;
}

int MyString::Length() {
	return int(str_size);
}
	
bool MyString::IsEqual(const MyString& other) {
	if (other.str_size != this->str_size) return false;
	for (size_t i = 0; i < this->str_size; i++) {
		if (*(this->str_data + i) != *(other.str_data + i))
			return false;
	}
	return true;
}

MyString MyString::Insert(const MyString& other, int start) {
	if (start<0 || start > this->str_size) return MyString("잘못된 인덱스입니다.");
		
	size_t result_size = this->str_size + other.str_size;

	char* temp = new char[result_size+1];

	for (size_t i = 0; i < start; i++) {
		temp[i] = *(this->str_data + i);
	}
	for (size_t j = 0; j < other.str_size; j++) {
		temp[start + j] = *(other.str_data + j);
	}
	for (size_t k = 0; k < this->str_size-start; k++) {
		temp[k+start+other.str_size] = *(this->str_data + k +start);
	}
	temp[result_size] = '\0';
	MyString result(temp);
	delete[] temp;

	return result;
}

MyString MyString::Substr(size_t start, size_t keep) {
	if (start < 0 || start >= this->str_size) return MyString("잘못된 인덱스입니다.");
	if (this->str_size < start + keep) return MyString("잘못된 파라미터입니다.");
	char* temp = new char[keep+1];
	for (size_t i = start; i < start + keep; i++) {
		temp[i - start] = *(this->str_data + i);
	}
	temp[keep] = '\0';
	MyString result(temp);
	delete[] temp;
	return result;
}

MyString MyString::Concat(const MyString& other) {
	size_t result_size = this->str_size + other.str_size;
	char* temp = new char[result_size + 1];

	for (size_t i = 0; i < this->str_size; i++) {
		temp[i] = this->str_data[i];
	}
	for (size_t j = 0; j < other.str_size; j++) {
		temp[j + this->str_size] = other.str_data[j];
	}
	temp[result_size] = '\0';
	MyString result(temp);
	delete[] temp;
	return result;
}
```
