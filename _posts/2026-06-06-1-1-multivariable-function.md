## 1. 다변수 함수 (Function of Several Variables)

**[KR]** 다변수 함수란 여러 개의 독립 변수를 입력으로 받아 단 하나의 실수 결괏값을 반환하는 수학적 관계를 뜻합니다.
**[EN]** A multivariable function takes multiple independent variables as inputs and maps them to a single real-valued output.

$$
f: \mathbb{R}^n \rightarrow \mathbb{R}
$$

$$
(x_1, x_2, \dots, x_n) \mapsto f(x_1, x_2, \dots, x_n)
$$

---

## 2. 최대 정의역과 치역 (Maximal Domain and Range)

### 최대 정의역 (Maximal Domain)
**[KR]** 주어진 함수식에 대입했을 때 유효한 실수 값이 계산될 수 있는 $\mathbb{R}^n$ 공간 내의 모든 입력점들의 집합을 최대 정의역이라고 부릅니다.
**[EN]** The maximal domain is the complete set of all input points in $\mathbb{R}^n$ that yield a valid, well-defined real number when plugged into the formula.

### 치역 (Range)
**[KR]** 정의역 안의 모든 점들을 함수에 대입하여 얻어지는 모든 가능한 출력값(결과)들의 모임을 치역이라고 합니다.
**[EN]** The range is the collection of all possible output values generated by the function when evaluated at every point within its domain.

$$
R = \{ f(x_1, x_2, \dots, x_n) \vert (x_1, x_2, \dots, x_n) \in D \}
$$

---

## 3. 시각화: 그래프와 등위선 (Visualization)

### 함수의 그래프 (Graphs)
**[KR]** 함수의 그래프는 정의역의 원소와 그에 대응하는 함숫값을 좌표로 묶어 공간 상에 시각화한 점들의 집합입니다.
**[EN]** A graph visually represents a function by plotting the set of all input coordinates alongside their corresponding output values in a coordinate space.

* **일변수 함수 (Function of one variable):**
  $$ \{ (x, y) \vert x \in D \text{ and } y = f(x) \} $$
* **이변수 함수 (Function of two variables):**
  $$ \{ (x, y, z) \vert (x, y) \in D \text{ and } z = f(x, y) \} $$

### 등위선과 등고선 지도 (Level Curves and Contour Maps)
**[KR]** 이변수 함수에서 함숫값이 특정한 상수 $k$와 같아지는 점들을 $xy$-평면 위에 이은 선을 등위선(Level curve)이라고 합니다. 이러한 선들을 여러 개 겹쳐 그린 도면을 등고선 지도(Contour map)라고 부릅니다.
**[EN]** A level curve is formed by connecting all points in the $xy$-plane that produce the same constant output value $k$. Displaying multiple level curves together creates what is known as a contour map.

$$
f(x, y) = k \quad (\text{where } k \text{ is a constant in the range})
$$
